# Quick Start: Print & PDF Export

## 🚀 How to Use (Quick Guide)

### Download Quotation as PDF
1. View any quotation
2. Click **"Download PDF"** (orange button)
3. PDF downloads automatically ✓

### Print Quotation
1. View any quotation  
2. Click **"Print A4"** (green button)
3. Select "Save as PDF" in printer dropdown
4. Click Save ✓

### Share via WhatsApp
1. View any quotation
2. Click **"WhatsApp PDF"** button
3. Message template pre-filled
4. Send to client ✓

### Email Quotation
1. View any quotation
2. Click **"Email PDF"** button
3. Email template pre-filled
4. Send to client ✓

## 🐛 Troubleshooting

| Issue | Solution |
|-------|----------|
| PDF not downloading | Check downloads folder, disable popup blocker |
| Alignment incorrect | Use "Download PDF" not print dialog |
| Images missing | Ensure visible in web preview, check console |
| Button not clickable | Scroll to top, ensure button not covered |
| File too large | Normal for quotations with many images |
